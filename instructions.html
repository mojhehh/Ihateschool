How to install a Veltra bookmark

  1. Click the small square Copy button on a bookmark.
  2. Open your browser's bookmarks manager (right-click the bookmarks bar → "Add page" or "Add bookmark").
  3. In the URL field, paste the code you copied from Veltra (it should be a one-line bookmarklet or URL).
  4. Give it a name and save. Click the bookmark from your bar to run it.

Mobile browsers sometimes block bookmarklets — desktop browsers are easiest.
